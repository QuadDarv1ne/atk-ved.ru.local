# Иконки и Favicon

## Текущие файлы

- `favicon.svg` - SVG favicon (создан)

## Необходимо создать

Для полной поддержки всех устройств и браузеров нужно создать следующие файлы:

### Favicon
- `favicon.ico` - 16x16, 32x32, 48x48 (для старых браузеров)
- `favicon-16x16.png` - 16x16
- `favicon-32x32.png` - 32x32

### Apple Touch Icons
- `apple-touch-icon.png` - 180x180

### PWA Icons
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Как создать иконки

### Вариант 1: Онлайн-генераторы
1. **RealFaviconGenerator** (https://realfavicongenerator.net/)
   - Загрузите логотип компании (минимум 512x512px)
   - Сервис автоматически создаст все необходимые размеры
   - Скачайте архив и распакуйте в эту папку

2. **Favicon.io** (https://favicon.io/)
   - Можно создать из текста, изображения или эмодзи
   - Генерирует основные размеры

### Вариант 2: Вручную (Photoshop/GIMP)
1. Создайте квадратное изображение 512x512px
2. Используйте логотип компании или букву "A"
3. Фон: #e31e24 (красный цвет компании)
4. Текст/логотип: белый (#ffffff)
5. Экспортируйте в нужные размеры

### Вариант 3: ImageMagick (командная строка)
```bash
# Из SVG создать PNG разных размеров
convert favicon.svg -resize 16x16 favicon-16x16.png
convert favicon.svg -resize 32x32 favicon-32x32.png
convert favicon.svg -resize 180x180 apple-touch-icon.png
convert favicon.svg -resize 192x192 icon-192x192.png
convert favicon.svg -resize 512x512 icon-512x512.png

# Создать .ico файл
convert favicon-16x16.png favicon-32x32.png favicon.ico
```

## Текущий дизайн

Используется простой дизайн:
- Красный фон (#e31e24)
- Белая буква "A" (первая буква АТК)
- Скруглённые углы (border-radius: 20%)

Можно заменить на:
- Полный логотип компании
- Стилизованную иконку грузовика/самолёта
- Комбинацию букв "АТК"

## Проверка

После создания иконок проверьте:
1. Отображение в браузере (вкладка)
2. Добавление на главный экран (мобильные)
3. PWA установка
4. Разные темы (светлая/тёмная)

## Оптимизация

- PNG файлы сжимайте через TinyPNG или ImageOptim
- SVG минифицируйте через SVGO
- ICO файл должен быть < 15KB
