## Как добавить изображения

### Способ 1: Через FTP
1. Подключитесь к серверу по FTP
2. Перейдите в папку `wp-content/themes/atk-ved/images/`
3. Загрузите файлы изображений
4. Убедитесь, что имена файлов совпадают с указанными выше

### Способ 2: Через файловый менеджер хостинга
1. Войдите в панель управления хостингом
2. Откройте файловый менеджер
3. Перейдите в `wp-content/themes/atk-ved/images/`
4. Загрузите файлы

### Способ 3: Через WordPress Media Library
1. Загрузите изображения в медиатеку WordPress
2. Скопируйте URL изображений
3. Обновите пути в `index.php`:
```php
<img src="<?php echo get_template_directory_uri(); ?>/images/hero-boxes.jpg" alt="...">
```

## Проверка

После загрузки изображений:
1. Откройте сайт в браузере
2. Проверьте, что изображения отображаются
3. Откройте консоль браузера (F12)
4. Проверьте, что нет ошибок 404
