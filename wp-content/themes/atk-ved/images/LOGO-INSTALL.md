# Инструкция по установке логотипа

## Шаг 1: Сохраните изображение логотипа

Сохраните изображение логотипа (которое вы прислали) как **logo.png**

## Шаг 2: Загрузите файл на сервер

### Способ 1: Через FTP
1. Подключитесь к серверу по FTP (FileZilla, WinSCP и т.д.)
2. Перейдите в папку: `wp-content/themes/atk-ved/images/`
3. Загрузите файл `logo.png` в эту папку

### Способ 2: Через файловый менеджер хостинга
1. Войдите в панель управления хостингом (cPanel, ISPmanager и т.д.)
2. Откройте файловый менеджер
3. Перейдите в папку: `wp-content/themes/atk-ved/images/`
4. Нажмите "Загрузить" и выберите файл `logo.png`

### Способ 3: Через WordPress админку
1. Войдите в админку WordPress
2. Перейдите в **Медиафайлы → Добавить новый**
3. Загрузите файл `logo.png`
4. Скопируйте URL загруженного файла
5. Откройте файл `header.php` через FTP
6. Замените путь в строке:
```php
<img src="<?php echo get_template_directory_uri(); ?>/images/logo.png" alt="АТК-ВЭД">
```
на:
```php
<img src="СКОПИРОВАННЫЙ_URL" alt="АТК-ВЭД">
```

## Шаг 3: Проверьте результат

1. Откройте сайт в браузере
2. Обновите страницу (Ctrl+F5 или Cmd+Shift+R)
3. Логотип должен отображаться в header

## Если логотип не отображается

### Проблема: Изображение не загружается (404 ошибка)

**Решение:**
1. Проверьте, что файл называется точно `logo.png` (не Logo.png, не logo.PNG)
2. Проверьте, что файл находится в правильной папке: `wp-content/themes/atk-ved/images/`
3. Проверьте права доступа к файлу (должно быть 644)

### Проблема: Логотип слишком большой или маленький

**Решение:**
Откройте файл `style.css` и измените высоту:
```css
.logo-link img {
    height: 40px; /* Измените это значение */
}
```

### Проблема: Логотип размытый

**Решение:**
1. Сохраните изображение в более высоком разрешении (2x или 3x)
2. Например, если нужен размер 200x40px, сохраните как 400x80px или 600x120px
3. В коде укажите реальный размер:
```html
<img src="logo.png" alt="АТК-ВЭД" width="200" height="40">
```

## Альтернатива: Использование WordPress Customizer

Вместо ручной загрузки можно использовать встроенную функцию WordPress:

1. Войдите в админку WordPress
2. Перейдите в **Внешний вид → Настроить**
3. Откройте **Идентичность сайта**
4. Нажмите **Выбрать логотип**
5. Загрузите файл `logo.png`
6. Сохраните изменения

После этого логотип будет автоматически использоваться вместо стандартного.

## Текущие настройки

### Размеры логотипа:
- **Desktop:** высота 40px, ширина auto
- **Mobile:** высота 35px, ширина auto

### Путь к файлу:
```
wp-content/themes/atk-ved/images/logo.png
```

### Код в header.php:
```php
<img src="<?php echo get_template_directory_uri(); ?>/images/logo.png" 
     alt="АТК-ВЭД" width="200" height="40">
```

## Оптимизация логотипа

### Рекомендации:
1. **Формат:** PNG с прозрачным фоном
2. **Размер файла:** не более 50KB
3. **Разрешение:** 2x для retina дисплеев (400x80px)
4. **Оптимизация:** используйте TinyPNG.com для сжатия

### Создание retina версии:
1. Сохраните изображение в 2x размере как `logo@2x.png`
2. Обновите код в header.php:
```html
<img src="logo.png" 
     srcset="logo.png 1x, logo@2x.png 2x" 
     alt="АТК-ВЭД" width="200" height="40">
```

## Поддержка

Если возникли проблемы:
1. Очистите кэш браузера (Ctrl+F5)
2. Очистите кэш WordPress (если используете плагин кэширования)
3. Проверьте консоль браузера (F12) на наличие ошибок
4. Убедитесь, что путь к файлу правильный
