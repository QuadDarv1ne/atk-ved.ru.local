# Быстрый старт - Тема АТК ВЭД

## За 5 минут до запуска

### 1. Активируйте тему
Админка → Внешний вид → Темы → Активировать "АТК ВЭД"

### 2. Установите Contact Form 7
Плагины → Добавить новый → Найти "Contact Form 7" → Установить → Активировать

### 3. Создайте меню
Внешний вид → Меню → Создать меню → Добавить произвольные ссылки:
- #services (Услуги)
- #delivery (Доставка)  
- #steps (Этапы)
- #faq (FAQ)
- #reviews (Отзывы)
- #contact (Контакты)

Отметить "Главное меню" → Сохранить

### 4. Укажите контакты
Внешний вид → Настроить → Контакты → Заполнить телефон и email → Опубликовать

### 5. Загрузите изображения
Через FTP в папку `wp-content/themes/atk-ved/images/`:
- hero-containers.jpg (800x600px)
- container.jpg (600x400px)

### 6. Готово!
Откройте главную страницу сайта и проверьте результат.

## Что дальше?

- Отредактируйте тексты в файле `index.php`
- Добавьте свой логотип (Внешний вид → Настроить → Идентичность сайта)
- Настройте Google Maps (замените iframe в index.php)
- Измените цвета в `style.css` (найдите #e31e24)
