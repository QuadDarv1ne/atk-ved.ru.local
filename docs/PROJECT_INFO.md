# Информация о проекте

## Название проекта
Тема WordPress "АТК ВЭД" - Товары для маркетплейсов из Китая оптом

## Описание
Кастомная одностраничная тема (лендинг) для WordPress, созданная по дизайн-макету. Тема предназначена для компании, занимающейся поставками товаров из Китая для продажи на маркетплейсах.

## Технологии
- WordPress 5.0+
- PHP 7.4+
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (jQuery)
- Responsive Design

## Структура файлов

## Основные возможности

### Секции лендинга
1. Hero Section - главный экран с заголовком и статистикой
2. Services - сетка услуг (6 карточек)
3. Delivery - способы и сроки доставки
4. Steps - этапы сотрудничества (6 шагов)
5. FAQ - часто задаваемые вопросы (аккордеон)
6. Reviews - отзывы клиентов (4 карточки)
7. Contact - форма обратной связи
8. Map - Google Maps с местоположением

### Функционал
- ✅ Адаптивный дизайн (desktop, tablet, mobile)
- ✅ Плавная прокрутка к секциям (smooth scroll)
- ✅ Липкая шапка при прокрутке
- ✅ Мобильное бургер-меню
- ✅ Аккордеон для FAQ
- ✅ Анимация появления элементов
- ✅ Поддержка меню WordPress
- ✅ Поддержка логотипа
- ✅ Настройка контактов через Customizer
- ✅ Интеграция с Contact Form 7

## Техническая поддержка

При возникновении проблем:
1. Проверьте версию WordPress (должна быть 5.0+)
2. Проверьте версию PHP (должна быть 7.4+)
3. Отключите другие плагины для проверки конфликтов
4. Очистите кэш браузера и плагинов кэширования
5. Проверьте консоль браузера на ошибки JavaScript (F12)
