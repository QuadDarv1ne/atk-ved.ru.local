# Тема WordPress "АТК ВЭД"

Кастомная тема для сайта компании АТК ВЭД - товары для маркетплейсов из Китая оптом.

## Установка

1. Скопируйте папку `atk-ved` в директорию `wp-content/themes/`
2. Войдите в админ-панель WordPress
3. Перейдите в "Внешний вид" → "Темы"
4. Активируйте тему "АТК ВЭД"

## Настройка

### Меню
1. Перейдите в "Внешний вид" → "Меню"
2. Создайте новое меню
3. Добавьте пункты меню с якорями:
   - Услуги (#services)
   - Доставка (#delivery)
   - Этапы (#steps)
   - FAQ (#faq)
   - Отзывы (#reviews)
   - Контакты (#contact)
4. Назначьте меню в позицию "Главное меню"

### Контакты
1. Перейдите в "Внешний вид" → "Настроить"
2. Откройте раздел "Контакты"
3. Укажите телефон и email

### Логотип
1. Перейдите в "Внешний вид" → "Настроить"
2. Откройте раздел "Идентичность сайта"
3. Загрузите логотип

### Контактная форма
Установите плагин Contact Form 7 для работы формы обратной связи:
1. Перейдите в "Плагины" → "Добавить новый"
2. Найдите "Contact Form 7"
3. Установите и активируйте
4. Создайте форму с ID=1

### Изображения
Добавьте изображения в папку `wp-content/themes/atk-ved/images/`:
- hero-containers.jpg - изображение контейнеров для главного экрана
- container.jpg - изображение для секции доставки

## Структура темы

```
atk-ved/
├── style.css           # Основные стили
├── functions.php       # Функции темы
├── index.php          # Главная страница (лендинг)
├── header.php         # Шапка сайта
├── footer.php         # Подвал сайта
├── page.php           # Шаблон страницы
├── single.php         # Шаблон записи
├── js/
│   └── main.js        # JavaScript
└── images/            # Изображения (создайте папку)
```

## Возможности

- Адаптивный дизайн
- Плавная прокрутка к секциям
- Липкая шапка
- Аккордеон для FAQ
- Анимация появления элементов
- Настройка контактов через Customizer
- Поддержка меню
- Поддержка логотипа

## Секции лендинга

1. Hero - главный экран с заголовком и статистикой
2. Услуги - сетка из 6 карточек услуг
3. Доставка - информация о способах доставки
4. Этапы сотрудничества - 6 шагов работы
5. FAQ - часто задаваемые вопросы
6. Отзывы - отзывы клиентов
7. Контакты - форма обратной связи
8. Карта - Google Maps

## Требования

- WordPress 5.0+
- PHP 7.4+
- Contact Form 7 (для формы обратной связи)
